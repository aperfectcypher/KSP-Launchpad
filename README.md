# KSP-Launchpad
## THIS IS AN EARLY WORK IN PROGRESS

Firmware for the KSP-Launchpad board.

MCU is an ATmega2560.
## Features

* Dual 3-axis joystick for rotation and translation
* Linear potentiometers for throttle
* SAS LED
* RCS LED
* LED bars as ship ressources gauges
* LCD screen (128x32px) for ship status

## Schematics

The schematics are written using kicad.
The schematics are available in the `schematics` directory.
